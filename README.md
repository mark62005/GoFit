# GoFit

**GoFit** is a beginner-friendly workout planner that helps users kickstart their fitness journey with smart, personalized routines. Designed for simplicity and motivation, GoFit guides new users through easy-to-follow workout plans tailored to their goals and fitness levels.

## 🚀 Tech Stack

**Frontend:**

- React.js
- Next.js
- Tailwind CSS
- Shadcn UI
- TypeScript

**Backend:**

- Node.js
- Express.js
- TypeScript
- PlanetScale

**Other Tools:**

- Clerk (Authentication)
- OpenAI API (Workout generation logic)
- Git & GitHub (Version Control)
